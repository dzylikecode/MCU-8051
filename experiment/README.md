## tips

使用 Windows Terminal

- `Ctrl + Shift + T` 新建标签页
- `Ctrl + Shift + W` 关闭标签页
- `Alt + Shift + =` 竖直分屏
- `Ctrl + Tab` 切换标签页

可以分屏窗口来执行烧录程序
